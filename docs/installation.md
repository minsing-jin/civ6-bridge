# Installation

## Stable release

To install civ6-bridge, run this command in your terminal:

```sh
uv add civ6-bridge
```

Or if you prefer to use `pip`:

```sh
pip install civ6-bridge
```

## From source

The source files for civ6-bridge can be downloaded from the [Github repo](https://github.com/minsing-jin/civ6_bridge).

You can either clone the public repository:

```sh
git clone git://github.com/minsing-jin/civ6_bridge
```

Or download the [tarball](https://github.com/minsing-jin/civ6_bridge/tarball/main):

```sh
curl -OJL https://github.com/minsing-jin/civ6_bridge/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd civ6_bridge
uv pip install .
```
