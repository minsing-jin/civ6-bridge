# civ6-bridge

![PyPI version](https://img.shields.io/pypi/v/civ6-bridge.svg)
[![Documentation Status](https://readthedocs.org/projects/civ6-bridge/badge/?version=latest)](https://civ6-bridge.readthedocs.io/en/latest/?version=latest)

This project is a protocol between civ6 and programming language.

* PyPI package: https://pypi.org/project/civ6-bridge/
* Free software: MIT License
* Documentation: https://civ6-bridge.readthedocs.io.

## Features
- You can control Civilization VI from your Python scripts.
- You can connect to Civilization VI game and send commands or receive data.
- Easy to use API for interacting with Civilization VI.
- Supports multiple platforms (Windows, macOS, Linux).
- Extensible architecture for adding new features.
